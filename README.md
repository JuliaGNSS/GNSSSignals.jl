# Generate GNSS signals.

## Features

* GPS L1
* GPS L5

## Getting started

Install:
```julia-repl
julia> ]
pkg> add GNSSSignals
```

## Usage

```julia
using GNSSSignals
gpsl1 = GPSL1()
code_freq = 1023e3
code_phase = 4
sample_freq = 4e6
prn = 1
sampled_code = gen_code.(Ref(gpsl1), 1:4000, code_freq, code_phase, sample_freq, prn)
```
